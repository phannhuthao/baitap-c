What is CSS?

Cascading style sheet (CSS) is used to format the layout of a webpage

With CSS, you can control the color, front, the size of text, the spacing between elements are positioned and laid out, What background images or background colors are to be used, different displays for different devices and screen sizes, and much move!

Tip: The word casading means that a style applied to a parent elements will also apply to all children elements within the parent. So, if you set the color of the body text to "Blue", all headings, paragraphs, and other text elements within the body will also get the same color(unless you specify something else)!
